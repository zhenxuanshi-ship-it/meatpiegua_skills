### Step 2: 技术指标全方位分析（核心！）

#### 一、均线系统（多周期共振）

**核心逻辑**：均线是趋势的最直接体现，多头排列=牛市，空头排列=熊市，缠绕=震荡。

| 均线 | 周期 | 作用 | 判断标准 |
|------|------|------|----------|
| 5日线 | 超短期 | 短线强弱 | 股价在5日线上=强势，跌破=转弱 |
| 10日线 | 短期 | 短期趋势 | 10日线方向向上=短期多头，向下=空头 |
| 20日线 | 中期 | 波段生命线 | 20日线是中期牛熊分界线 |
| 60日线 | 中长期 | 季度趋势 | 60日线向上=中期向好，向下=中期走坏 |
| 120日线 | 长期 | 半年线 | 牛熊转换的重要参考 |
| 250日线 | 超长期 | 年线 | 年线之上=牛市，之下=熊市 |

**多头排列**（强势上涨）：5日 > 10日 > 20日 > 60日，所有均线向上发散
**空头排列**（弱势下跌）：5日 < 10日 < 20日 < 60日，所有均线向下发散
**均线缠绕**（震荡整理）：各均线粘合在一起，即将选择方向

**均线计算公式（验证用）**：
```
简单移动平均线（SMA）= (P1 + P2 + ... + PN) / N

其中：
- P1, P2, ... PN = 最近N个交易日的收盘价
- N = 均线周期（5、10、20、60、120、250）

示例（5日均线）：
最近5日收盘价分别为：10元、10.5元、11元、10.8元、11.2元
5日均线 = (10 + 10.5 + 11 + 10.8 + 11.2) / 5 = 10.7元
```

**均线数据验证 checklist**：
- [ ] 获取的均线数值是否在合理区间（与当前股价相差不超过±20%）
- [ ] 短期均线（5日、10日）是否更接近当前股价
- [ ] 长期均线（60日、120日）是否更平滑、变化更慢
- [ ] 均线排列状态是否与股价走势一致

**数据获取方式优先级**：
1. **优先使用Web搜索获取的现成均线数据**
2. **若无法获取精确值，可通过近期K线走势推断均线位置**
3. **所有均线判断必须基于最新1-2个交易日的数据**

**金叉/死叉判断**：
- **金叉**：短期均线上穿长期均线，买入信号
  - 5日上穿10日：短线金叉，超短机会
  - 10日上穿20日：波段金叉，中期机会
  - 20日上穿60日：趋势金叉，长线机会
- **死叉**：短期均线下穿长期均线，卖出信号
  - 5日下穿10日：短线死叉，止盈/止损
  - 10日下穿20日：波段死叉，减仓信号
  - 20日下穿60日：趋势死叉，清仓信号

#### 二、MACD指标（趋势动能）

**核心逻辑**：MACD看趋势动能和背离，是判断中期趋势的核心指标。

**指标构成**：
- **DIF（快线）**：12日EMA - 26日EMA
- **DEA（慢线）**：DIF的9日EMA
- **MACD柱（BAR）**：(DIF - DEA) × 2

**判断标准**：

| 状态 | DIF vs DEA | MACD柱 | 含义 |
|------|------------|--------|------|
| 强势多头 | DIF > DEA，且都在0轴上 | 红柱放大 | 上涨动能强劲 |
| 弱势多头 | DIF > DEA，但在0轴下 | 红柱缩小 | 反弹乏力，小心回调 |
| 强势空头 | DIF < DEA，且都在0轴下 | 绿柱放大 | 下跌动能强劲 |
| 弱势空头 | DIF < DEA，但在0轴上 | 绿柱缩小 | 调整接近尾声 |

**金叉/死叉**：
- **金叉**：DIF上穿DEA，看涨信号
  - 0轴上方金叉：强势延续，加仓点
  - 0轴下方金叉：反弹信号，谨慎参与
- **死叉**：DIF下穿DEA，看跌信号
  - 0轴上方死叉：调整开始，减仓
  - 0轴下方死叉：下跌趋势加速，清仓

**背离判断**（极其重要！）：
- **顶背离**（看跌）：股价创新高，但MACD不创新高（DIF/DEA/BAR都低于前高）
  - 说明上涨动能衰竭，随时可能暴跌
  - 这是必须卖出的信号！
- **底背离**（看涨）：股价创新低，但MACD不创新低（DIF/DEA/BAR都高于前低）
  - 说明下跌动能衰竭，可能反弹
  - 这是可以试仓的信号

**郑大姐解读**：
- "MACD红柱持续放大，多头牛逼，继续持有！"
- "顶背离！你眼睛瞎了？股价新高MACD不跟，这是要崩的节奏！"
- "0轴下方金叉，这只能算反弹，别当是反转！"

#### 三、RSI相对强弱指标（超买超卖）

**核心逻辑**：RSI衡量买卖双方力量对比，判断超买超卖状态。

**计算方法与周期**：

**标准周期**：14日（默认，也可使用6日或9日短线周期）

**计算公式**：
```
RSI = 100 - 100/(1 + RS)

其中：
RS = 最近N日平均涨幅 / 最近N日平均跌幅

平均涨幅 = (第1日涨幅 + 第2日涨幅 + ... + 第N日涨幅) / N  （只计算上涨日）
平均跌幅 = (第1日跌幅 + 第2日跌幅 + ... + 第N日跌幅) / N  （只计算下跌日，取绝对值）

注意：
- 上涨日：当日收盘价 > 前一日收盘价，涨幅为正值
- 下跌日：当日收盘价 < 前一日收盘价，跌幅为正值（取绝对值）
- 若平均跌幅为0，RSI = 100（全部上涨）

示例（14日RSI）：
最近14日：平均涨幅 = 2%，平均跌幅 = 1%
RS = 2/1 = 2
RSI = 100 - 100/(1+2) = 100 - 33.33 = 66.67
```

**判断标准**：

| RSI区间 | 状态 | 操作策略 |
|---------|------|----------|
| > 80 | 严重超买 | 随时可能回调，减仓/观望 |
| 70-80 | 超买 | 谨慎追高，考虑止盈 |
| 50-70 | 强势区 | 持有多单，回踩加仓 |
| 50 | 平衡 | 观望，等方向 |
| 30-50 | 弱势区 | 持有空单，反弹减仓 |
| 20-30 | 超卖 | 考虑抄底，分批建仓 |
| < 20 | 严重超卖 | 随时可能反弹，可以试仓 |

**顶背离/底背离**：
- RSI顶背离：股价创新高，RSI不创新高 → 看跌
- RSI底背离：股价创新低，RSI不创新低 → 看涨

#### 四、布林带（BOLL）—— 波动区间

**核心逻辑**：布林带看价格波动区间和突破，判断超买超卖的另一种方式。

**指标构成**：
- **中轨（MB）**：20日均线
- **上轨（UP）**：中轨 + 2倍标准差
- **下轨（DN）**：中轨 - 2倍标准差

**判断标准**：

| 位置 | 含义 | 策略 |
|------|------|------|
| 股价触及上轨 | 超买，可能回调 | 减仓/观望 |
| 股价在上轨上方 | 极度强势，但也可能透支 | 持有但警惕 |
| 股价在中轨上方 | 强势区 | 持有多单 |
| 股价在中轨附近 | 平衡状态 | 观望 |
| 股价在中轨下方 | 弱势区 | 持有空单 |
| 股价触及下轨 | 超卖，可能反弹 | 考虑抄底 |
| 股价在下轨下方 | 极度弱势，但也可能超跌 | 等企稳再进 |

**收口与开口**：
- **收口**：上轨与下轨距离缩小，波动率降低，即将选择方向
- **开口**：上轨与下轨距离扩大，波动率增加，趋势加强

#### 五、KDJ随机指标（短线之王）

**核心逻辑**：KDJ对价格波动反应灵敏，适合短线操作。

**指标构成与计算公式**：

**Step 1: 计算RSV（未成熟随机值）**
```
RSV = (当日收盘价 - 最近N日最低价) / (最近N日最高价 - 最近N日最低价) × 100

其中：
- N = 9（默认周期，也可使用5日或14日）
- 最近N日最高价：N个交易日内出现的最高价
- 最近N日最低价：N个交易日内出现的最低价

示例：
当日收盘价 = 50元
最近9日最高价 = 55元，最低价 = 45元
RSV = (50 - 45) / (55 - 45) × 100 = 50%
```

**Step 2: 计算K值（快线）**
```
当日K值 = 2/3 × 前一日K值 + 1/3 × 当日RSV

- 首日K值通常取50（或前9日RSV平均值）
- 平滑系数2/3是行业标准

示例：
前一日K值 = 60，当日RSV = 50
当日K值 = 2/3 × 60 + 1/3 × 50 = 40 + 16.67 = 56.67
```

**Step 3: 计算D值（慢线）**
```
当日D值 = 2/3 × 前一日D值 + 1/3 × 当日K值

- 首日D值通常取50（与K值相同）
- 平滑系数2/3是行业标准

示例：
前一日D值 = 55，当日K值 = 56.67
当日D值 = 2/3 × 55 + 1/3 × 56.67 = 36.67 + 18.89 = 55.56
```

**Step 4: 计算J线（方向线）**
```
J值 = 3 × 当日K值 - 2 × 当日D值

- J值放大K与D的差距，波动更剧烈
- J > 100：严重超买，J < 0：严重超卖

示例：
K值 = 56.67，D值 = 55.56
J值 = 3 × 56.67 - 2 × 55.56 = 170.01 - 111.12 = 58.89
```

**完整计算示例**：
```
假设某股票最近9日数据：
最高价 = 55元，最低价 = 45元，当日收盘价 = 50元
前一日K = 60，前一日D = 55

RSV = (50-45)/(55-45) × 100 = 50
K = 2/3 × 60 + 1/3 × 50 = 56.67
D = 2/3 × 55 + 1/3 × 56.67 = 55.56
J = 3 × 56.67 - 2 × 55.56 = 58.89
```

**判断标准**：

| 区间 | 状态 | 策略 |
|------|------|------|
| K、D、J > 80 | 超买区 | 随时准备卖出 |
| K、D、J > 100 | 严重超买 | 必须减仓 |
| K、D、J < 20 | 超卖区 | 随时准备买入 |
| K、D、J < 0 | 严重超卖 | 可以试仓 |

**金叉/死叉**：
- **金叉**：K线上穿D线，短线买入信号
  - 20以下金叉：超卖区反弹，可信度高
  - 80以上金叉：高位纠缠，可能是诱多
- **死叉**：K线下穿D线，短线卖出信号
  - 80以上死叉：超买区回调，可信度高
  - 20以下死叉：低位纠缠，可能是诱空

**J线极端值**：
- J > 100：严重超买，必回调
- J < 0：严重超卖，必反弹

#### 六、成交量分析（量价关系）

**核心逻辑**：量是价的先行指标，无量上涨都是耍流氓。

**量价配合原则**：

| 价格 | 成交量 | 含义 | 策略 |
|------|--------|------|------|
| 上涨 | 放量 | 量价齐升，健康 | 持有/加仓 |
| 上涨 | 缩量 | 量价背离，虚假上涨 | 警惕，准备减仓 |
| 下跌 | 放量 | 恐慌盘杀出，可能见底 | 观察，等企稳 |
| 下跌 | 缩量 | 抛压减轻，可能企稳 | 关注，等放量反弹 |
| 横盘 | 放量 | 资金介入，即将突破 | 关注突破方向 |
| 横盘 | 缩量 | 无人问津，继续磨 | 观望 |

**均量线判断**：
- 成交量 > 5日均量：放量
- 成交量 > 5日均量 > 10日均量：持续放量
- 成交量 < 5日均量：缩量

**特殊形态**：
- **天量**：成交量创近期新高，可能是顶部或底部信号
- **地量**：成交量创近期新低，可能是变盘前兆
- **堆量**：成交量持续放大，资金持续介入

- "放量下跌，恐慌盘出来了，离底不远了"

#### 七、动能分析（量价动能+资金流向）

**核心逻辑**：动能分析看资金进出的力度和持续性，判断趋势强度和转折点。

##### 1. 成交量动能（VMI）

**成交量变异率（VMR）**：
- 今日成交量 / 5日均量
- VMR > 1.5：放量，资金活跃
- VMR < 0.8：缩量，资金观望
- VMR > 2：天量，可能见顶/见底

**量价配合动能评分**：

| 价格走势 | 成交量变化 | 动能评分 | 含义 |
|----------|------------|----------|------|
| 上涨 | 放量（>5日均量150%） | +3 | 强多头动能 |
| 上涨 | 温和放量（110-150%） | +2 | 健康多头动能 |
| 上涨 | 平量（90-110%） | +1 | 动能减弱 |
| 上涨 | 缩量（<90%） | -1 | 量价背离，警惕 |
| 下跌 | 放量（>150%） | -3 | 强空头动能 |
| 下跌 | 缩量（<90%） | +1 | 抛压减轻，可能企稳 |

**成交量趋势**：
- 成交量5日线上穿10日线：量能增强
- 成交量5日线下穿10日线：量能萎缩
- 成交量连增3天：资金持续流入
- 成交量连缩3天：资金持续流出

**郑大姐解读**：
- "放量上涨+3分，动能强劲，可以跟！"
- "缩量上涨-1分，量价背离，这是假突破！"
- "放量下跌-3分，恐慌盘杀出，等等再说"

##### 2. 资金流向动能（Money Flow）

**大单资金流向**：
- 超大单（>100万）：主力机构动向
- 大单（20-100万）：大资金动向
- 中单（5-20万）：中小资金
- 小单（<5万）：散户动向

**资金动能判断**：

| 指标 | 计算方式 | 判断标准 |
|------|----------|----------|
| 主力净流入 | 超大单+大单净流入 | >5000万：强流入，<br>0-5000万：弱流入，<br><0：流出 |
| 散户净流入 | 中单+小单净流入 | 正：散户买入（警惕），<br>负：散户卖出（利好） |
| 资金集中度 | 主力净流入/成交额 | >20%：主力控盘，<br>10-20%：主力参与，<br><10%：散户行情 |
| 资金流向趋势 | 连续3日主力净流入 | 持续流入：趋势确立，<br>持续流出：趋势走坏 |

**五档盘口分析**：
- 买盘挂单 > 卖盘挂单2倍：买方强势
- 卖盘挂单 > 买盘挂单2倍：卖方强势
- 大单压盘（卖一挂大单）：主力压价吸筹
- 大单托单（买一挂大单）：主力托价出货

**郑大姐解读**：

- "卖一挂着万手大单，这是压盘吸筹呢"

##### 3. 价格动能指标

**变动率（ROC）**：

**标准周期**：12日（默认，也可使用25日）

**计算公式**：
```
ROC = (今日收盘价 - N日前收盘价) / N日前收盘价 × 100%

其中：
- N = 12（默认周期）
- N日前收盘价 = 12个交易日前的收盘价

示例：
今日收盘价 = 55元
12日前收盘价 = 50元
ROC = (55 - 50) / 50 × 100% = 10%
```

**判断标准**：
- ROC > 10%：短期强势
- ROC < -10%：短期弱势
- ROC从负转正：动能转强
- ROC从正转负：动能转弱

---

**动量指标（MTM）**：

**标准周期**：10日（默认，也可使用12日或20日）

**计算公式**：
```
MTM = 今日收盘价 - N日前收盘价

其中：
- N = 10（默认周期）
- 单位：元（价格差值，非百分比）

示例：
今日收盘价 = 55元
10日前收盘价 = 52元
MTM = 55 - 52 = 3元
```

**判断标准**：
- MTM > 0：上涨动能（当前价格高于N日前）
- MTM < 0：下跌动能（当前价格低于N日前）
- MTM = 0：价格回到N日前水平
- MTM背离：警惕反转（价格创新低但MTM不创新低）

**郑大姐解读**：
- "ROC从-5跳到+8，动能转强，可以考虑"
- "MTM连续3天下降，上涨动能衰竭"

##### 4. 筹码动能（换手率分析）

**换手率分析**：
- 换手率 < 3%：冷清，无资金关注
- 换手率 3-7%：正常交易
- 换手率 7-15%：活跃，资金关注
- 换手率 > 15%：过热，警惕变盘

**量比分析**：
- 量比 = 当日成交量 / 前5日平均成交量
- 量比 < 0.8：缩量
- 量比 0.8-1.2：正常
- 量比 1.2-2：放量
- 量比 > 2：天量

**筹码交换率**：
- 连续3日换手率递增：筹码活跃，资金博弈
- 高位高换手：主力出货
- 低位高换手：主力吸筹
- 缩量涨停：筹码锁定好，可能连板

**郑大姐解读**：

##### 6. 动能背离分析（重要！）

**价格与资金背离**：
- 价格上涨 + 主力资金流出 = 假涨出货 → **看跌**
- 价格下跌 + 主力资金流入 = 假跌吸筹 → **看涨**

**动能指标背离**：
- MACD/RSI/KDJ与价格背离（见前述）
