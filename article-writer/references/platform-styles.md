# 各平台写作风格指南

## 小红书

### 核心特征
- **种草感强**：分享真实体验，像朋友推荐
- **视觉导向**：emoji丰富，分段短，适合手机阅读
- **情绪价值**：提供情绪共鸣，而非纯信息

### 格式规范
- 标题：12-20字，带emoji，制造悬念或利益点
- 段落：每段2-4行，多用换行
- 符号：善用「」、『』、✨、🔥、💡、❗️、👉
- 结构：痛点引入 → 解决方案 → 效果展示 → 行动号召

### 语言风格
- 人称：多用"姐妹们"、"宝子们"、"我"
- 语气：亲切、热情、略带夸张
- 词汇："绝绝子"、"YYDS"、"挖到宝"、"踩雷"、"亲测"

### 示例开头
```
姐妹们！我发现了一个神器！😱

真的不夸张，用了三个月直接省下一个包的钱！💰

之前我也是抱着试试看的心态...
```

### 结尾模板
```
觉得有用的话点个赞👍
还有什么问题评论区问我～
#话题标签 #话题标签
```

## 公众号

### 核心特征
- **深度内容**：适合2000-5000字长文
- **故事性强**：用故事包装观点
- **传播导向**：金句多，易转发

### 格式规范
- 标题：引发好奇或情绪共鸣，可长可短
- 段落：4-6行为一段，避免大段文字
- 重点：关键句加粗、变色或单独成段
- 结构：钩子开头 → 故事展开 → 观点升华 → 金句结尾

### 语言风格
- 人称：可用"我"或全知视角
- 语气：成熟、有观点、带态度
- 技巧：设置悬念、反转、对比

### 排版技巧
- 引用名人名言
- 关键数据突出显示
- 适当使用分隔线
- 文末引导转发/在看

### 示例结构
```markdown
# 标题：那个35岁被裁的中年人，后来怎么样了

> 成年人的世界，没有容易二字。

（故事开头，设置场景）

去年冬天，我在咖啡馆遇到一个...

（故事发展，制造冲突）

他当时说了一句话，我至今记得...

（观点输出）

其实，35岁危机的本质是...

（金句结尾）

人生没有白走的路，每一步都算数。

---

如果这篇文章对你有启发，欢迎转发给需要的朋友。
```

## 知乎

### 核心特征
- **专业可信**：引用数据、文献、案例
- **逻辑严密**：论点-论据-论证完整
- **信息密度高**：干货多，废话少

### 格式规范
- 标题：直接点明问题，如"如何看待XXX？"
- 结构：总分总，多层分级标题
- 引用：标注来源，增加可信度
- 排版：有序/无序列表清晰呈现

### 语言风格
- 人称：客观为主，可用第一人称"我的经历"
- 语气：理性、克制、有说服力
- 禁忌：避免过于情绪化、避免软广告感

### 写作技巧
- 开头直接回答核心问题
- 分点论述，每点有论据支撑
- 承认反方观点并反驳
- 结尾总结升华

### 示例结构
```markdown
## 先说结论：XXX的核心原因在于...

### 一、背景分析

根据XX数据显示...

### 二、核心原因

**1. 原因A**
- 论据1
- 论据2

**2. 原因B**
...

### 三、实际影响
...

### 四、我的建议
...

---

以上。有相关问题欢迎评论区交流。
```

## X / Twitter

### 核心特征
- **短小精悍**：280字以内（中文约140字）
- **观点鲜明**：一句有力的话
- **传播性强**：易转发、易讨论

### 格式规范
- 单条或线程（thread）
- 善用 @ # 
- 可配图增加传播

### 语言风格
- 直接、犀利、有态度
- 可用反问、对比制造张力
- 适当使用网络用语

### 示例
```
很多人问：35岁学编程晚吗？

我的回答：不晚，但你要做好三年赚不到钱的准备。

大部分人说"想转行"，其实只想"想"。

真正去做的，从来不会问晚不晚。

#编程 #转行 #35岁
```

## 掘金

### 核心特征
- **技术导向**：代码、命令、配置
- **实操性强**：步骤清晰，可复制
- **专业术语**：准确使用技术名词

### 格式规范
- 代码块标注语言
- 步骤编号清晰
- 报错信息完整呈现
- 环境配置说明清楚

### 语言风格
- 技术准确，不夸大
- 承认技术的局限性
- 分享踩坑经验

### 示例结构
```markdown
# 标题：手把手教你用Vite搭建Vue3项目

## 前言

最近在公司项目中...

- Node.js >= 16
- npm >= 8

## 步骤一：创建项目

```bash
npm create vite@latest my-app --template vue
```

## 步骤二：安装依赖

...

## 踩坑记录

### 坑1：xxx
原因：xxx
解决：xxx

## 总结
...
```

## B站专栏

### 核心特征
- **年轻化**：Z世代语言，玩梗
- **二次元元素**：表情包、颜文字
- **互动性强**：引导弹幕、评论

### 语言风格
- "兄弟们"、"家人们"
- "这波"、"属实"、"蚌埠住了"
- 表情符号：(ಡωಡ) (╯°□°）╯

### 示例
```
兄弟们，今天来聊个离谱的事...

这波操作真的给我看傻了😂

有一说一，确实...
```

## 头条号

### 核心特征
- **标题党**：悬念、数字、情绪词
- **信息量大**：适合中老年读者
- **正能量**：避免过于负面

### 标题公式
- 数字+利益："3个方法让你月入过万"
- 悬念+情绪："看完这个，我沉默了..."
- 对比+冲突："富人思维和穷人思维的区别"

### 语言风格
- 通俗易懂，避免专业术语
- 多用短句
- 适当煽情
