# 反AI检测写作技巧

## 为什么需要反AI检测

各平台（知乎、公众号、头条等）正在加强对AI生成内容的识别和限制。被识别为AI内容会导致：
- 限流/降权
- 无法推荐
- 账号风险提示

## 核心原则：让文字有"人味"

### 一、打破AI的典型语言模式

#### ❌ AI常见痕迹

**过度结构化的连接词**
```
首先...其次...最后...
综上所述...
值得注意的是...
从以上分析可以看出...
```

**过于工整的句式**
```
人工智能是当下的热门话题，它在各个领域都有着广泛的应用。
随着科技的发展，人们的生活方式发生了巨大的变化。
```

**每段长度一致**
AI倾向于生成段落长度相似的文本，视觉上非常规整。

**过度使用排比**
```
它让我们更便捷，让我们更高效，让我们更智能。
```

#### ✅ 人类写作特征

**口语化连接**
```
说实话，...
其实吧，...
你知道吗，...
讲真，...
怎么说呢，...
```

**情绪化的插入语**
```
（这点真的很重要）
——别问我怎么知道的
[笑哭]
（此处省略一万字吐槽）
```

**不完美的表达**
```
我也不知道怎么说清楚，大概就是这么个意思...
反正你用了就知道，真的，不骗你。
```

**突然变短的句子**
```
就是这么简单。
真的。
不骗你。
```

### 二、加入个人元素

**具体的时间地点**
```
上周三下午，我在咖啡厅...
去年冬天，我遇到一个人...
```

**真实的困惑和纠结**
```
说实话，我一开始也是拒绝的...
当时我特别纠结，不知道该不该...
```

**感官细节**
```
我记得那天特别冷，风吹得脸疼...
屏幕的光刺得我眼睛难受...
```

**个人偏好和偏见**
```
我个人不太喜欢这种方式，因为...
说实话，我对这个东西是有偏见的...
```

### 三、制造"瑕疵"

**允许语法不完美**
```
这个东西吧，怎么说呢，反正就挺好用的。
（虽然语法不太对，但口语就是这样）
```

**偶尔跑题**
```
说到这个，我想起另一件事...
（说完再拉回来）
不过回到正题...
```

**重复和啰嗦**
```
真的，真的特别好用。
重要的事情说三遍：不要买，不要买，不要买。
```

**括号补充**
```
我当时用了一个月（其实也就三个星期），效果真的明显。
```

### 四、平台特定的反检测技巧

#### 知乎
- 开头直接给结论，不绕弯子
- 承认"我不知道"或"这部分我不确定"
- 加入个人经历段落
- 引用具体数据时标注来源（哪怕是模糊的）

#### 公众号
- 故事开头要有具体场景
- 情绪起伏要自然（不要一直正能量）
- 金句要"土"一点，不要太文艺
- 结尾不要太圆满，留有余味

#### 小红书
- emoji不要太多，也不要太整齐
- 口语化词汇要自然（"绝绝子"用多了也假）
- 加入"踩雷"经历，不要全是好评
- 数字要具体（"省了300块"而不是"省了很多"）

#### 掘金/技术社区
- 代码报错要真实（贴真实报错信息）
- 承认"这个方法有局限性"
- 分享"我踩过的坑"
- 版本号要具体

### 五、检测自查清单

写完文章后，检查以下项目：

- [ ] 是否有"首先/其次/最后"三连？
- [ ] 每段长度是否都差不多？
- [ ] 是否有过度的排比句？
- [ ] 是否有"综上所述"类总结词？
- [ ] 是否有具体的个人经历？
- [ ] 是否有口语化表达？
- [ ] 是否有情绪起伏？
- [ ] 是否有短句/单字段落？

### 六、改写示例

#### 原文（AI感强）
```
人工智能正在改变我们的生活。首先，在医疗领域，AI可以辅助诊断疾病；其次，在教育领域，AI可以提供个性化学习方案；最后，在交通领域，自动驾驶技术正在成熟。综上所述，AI的影响是全方位的。
```

#### 改写后（有人味）
```
说实话，AI这东西已经渗透到我们生活的方方面面了。

我一个朋友是医生，他跟我说现在看片子的效率提升了不少——当然他也吐槽说，有时候AI给出的建议挺离谱的，还得人工把关。

教育这块也是。我侄女现在用的学习APP，能根据她错题的情况推送练习。效果嘛...说实话，我觉得是把双刃剑。

最离谱的是自动驾驶。上周我打了个车，师傅跟我说他一直在关注这个，"说不定哪天就失业了"。

反正这东西，来都来了，躲是躲不掉的。
```

## 记忆口诀

**三有三无：**
- 有瑕疵、有情绪、有细节
- 无套话、无工整、无总结
