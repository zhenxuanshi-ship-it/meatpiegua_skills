| **周变化** | 与上周同日比 | 中期趋势 |
| **月变化** | 与上月同日比 | 宏观趋势 |
| **关键位** | 与支撑/压力位比 | 技术研判 |
| **相关性** | 各资产联动变化 | 资金流向 |

**重点关注信号**：
- 🚨 **美元与黄金同涨** → 地缘风险升级

## 典型映射案例

### 案例1：美元暴涨+黄金同涨
**事件**：地缘冲突升级，避险资金涌入美元和黄金
**研判**：全球进入risk-off模式，资金逃离风险资产
**A股映射**：黄金股（山东黄金、赤峰黄金）受益；出口型股票（汇率贬值受益）

### 案例2：美债收益率倒挂收窄
**事件**：10Y-2Y利差从负转正
**研判**：衰退担忧降温，经济软着陆预期升温
**A股映射**：成长股（新能源、科技）受益；银行板块受益（息差扩大）

### 案例3：比特币与纳指背离
**事件**：纳指下跌但比特币上涨
**研判**：加密市场独立行情，资金从传统风险资产溢出到加密
**A股映射**：区块链概念股（ if any）；关注A股风险偏好变化

### 案例4：日元突然暴涨
**事件**：USD/JPY从150跌破140
**研判**：日本央行干预或套息交易平仓，全球流动性收紧
**A股映射**：警惕外资流出压力；高估值成长股承压

### 案例5：人民币破7.3
**事件**：USD/CNY突破7.3心理关口
**研判**：资本外流压力加大，央妈可能干预
**A股映射**：出口型企业受益；外资重仓股承压；黄金股受益（保值需求）
