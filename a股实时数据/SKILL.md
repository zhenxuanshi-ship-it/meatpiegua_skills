# A股实时数据查询

## 数据来源
- 新浪财经（免费，15分钟延迟）
- 腾讯财经（免费，15分钟延迟）
- AKShare（Python库，免费）

## 使用方法

### 查询个股实时行情
```bash
# 查询单只股票
/root/.openclaw/workspace/skills/a股实时数据/stock-query.sh sz002340

# 查询多只股票
/root/.openclaw/workspace/skills/a股实时数据/stock-query.sh sz002340,sh601857,sz300102
```

**代码前缀规则：**
- `sh` 开头：上海主板（600/601/603/688）
- `sz` 开头：深圳主板/创业板（000/002/300）
- `bj` 开头：北京交易所（430/830/87）

### 快速查询命令别名
添加到 `~/.bashrc`：
```bash
alias stock='/root/.openclaw/workspace/skills/a股实时数据/stock-query.sh'
```

然后直接：
```bash
stock sz002340
stock sh601857
```
