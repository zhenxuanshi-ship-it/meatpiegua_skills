# 知乎文章发布参考指南

## 知乎文章格式规范

### 标题要求
- 长度：1-100字
- 不能包含特殊符号：`<>/\|:*?"`
- 建议包含关键词，提高搜索排名
- 标题党慎用，避免被限流

### 正文要求
- 最少100字，建议1000-3000字
- 支持格式：
  - 一级/二级/三级标题
  - 加粗、斜体
  - 有序/无序列表
  - 引用块
  - 链接
  - 图片
  - 表格
  - 代码块

### 封面图要求
- 尺寸：建议 900 x 500 像素
- 格式：JPG、PNG
- 大小：不超过5MB
- 建议：与内容相关，视觉吸引力强

### 话题标签
- 数量：2-5个
- 选择：与内容高度相关的话题
- 热门话题：增加曝光机会
- 细分领域：精准触达目标读者

## Markdown转知乎格式对照表

| Markdown语法 | 知乎显示效果 | 备注 |
|-------------|-------------|------|
| `# 标题` | 一级标题 | 字号最大 |
| `## 标题` | 二级标题 | 常用章节标题 |
| `### 标题` | 三级标题 | 小节标题 |
| `**加粗**` | **加粗** | 强调重点 |
| `*斜体*` | *斜体* | 次要强调 |
| `~~删除线~~` | ~~删除线~~ | 表示过时内容 |
| `- 列表` | • 列表 | 无序列表 |
| `1. 列表` | 1. 列表 | 有序列表 |
| `> 引用` | 引用块 | 引用他人观点 |
| `[文字](链接)` | 文字链接 | 外部链接 |
| `![描述](图片URL)` | 图片 | 需上传图床 |
| `` `代码` `` | `代码` | 行内代码 |
| ````语言\n代码\n```` | 代码块 | 支持语法高亮 |
| `---` | 分隔线 | 内容分段 |
| `| 表头 |` | 表格 | 需特殊处理 |

## 文章发布流程

### 方式一：Web端发布
1. 登录知乎账号
2. 进入创作中心
3. 点击"写文章"
4. 粘贴正文内容
5. 设置封面图
6. 添加话题标签
7. 预览并发布

### 方式二：API发布（需申请）
```python
# 伪代码示例
import requests

def publish_zhihu_article(title, content, tags, cover_image):
    url = "https://www.zhihu.com/api/v4/articles"
    headers = {
        "Authorization": "Bearer YOUR_TOKEN",
        "Content-Type": "application/json"
    }
    data = {
        "title": title,
        "content": content,
        "topics": tags,
        "cover_image": cover_image
    }
    response = requests.post(url, json=data, headers=headers)
    return response.json()
```

## 内容优化建议

### 标题优化
✅ 好的标题：
- "2026年投资展望：AI产业的10大机会"
- "美联储暂停降息，对A股意味着什么？"
- "我花了10万学费，总结出的投资经验"

❌ 差的标题：
- "大家好"
- "今天说个事"
- "震惊！你绝对不知道的秘密"

### 结构优化
1. **开头**：用钩子吸引读者（问题、数据、故事）
2. **中间**：分段清晰，每段一个观点
3. **结尾**：总结+互动（提问、邀请评论）
4. **配图**：3-5张，每500字配一张图

### SEO优化
- 标题包含核心关键词
- 正文自然融入关键词
- 添加相关话题标签
- 使用知乎站内链接

## 常见问题

**Q: 文章可以修改吗？**
A: 可以，发布后仍可编辑，但修改后可能需要重新审核。

**Q: 文章被限流怎么办？**
A: 检查是否有敏感词、标题党、广告内容。修改后重新发布。

**Q: 可以发布重复内容吗？**
A: 不建议，知乎对重复/搬运内容会降权处理。

**Q: 文章可以删除吗？**
A: 可以，在个人主页-创作内容中删除。

**Q: 如何查看文章数据？**
A: 创作中心-内容管理-数据分析，可查看阅读、点赞、评论等数据。
