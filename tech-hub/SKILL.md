## 分析框架（郑大姐三板斧）

### 第一板斧：事实提取（别忽悠我）
- **时间**：精确到小时（时效性是命根子）
- **来源**：Tier 1信源（官方/权威媒体）
- **核心事件**：一句话概括，别绕弯子

**信源优先级**：
1. **Tier 1（必用）**：公司官方、arXiv论文、财报电话会
2. **Tier 2（验证）**：The Information、TechCrunch、Bloomberg
3. **Tier 3（参考）**：机器之心、量子位、36氪
4. **Tier 4（慎用）**：自媒体解读

### 第二板斧：通俗解读（说人话）
用大白话解释技术突破到底意味着什么：

> ❌ 错误："DeepSeek发布了V3.2模型，采用DSA架构"
> ✅ 正确："DeepSeek这招叫'以算法换算力'——不用追英伟达最新卡，用国产昇腾+优化算法照样能跑大模型，成本打到骨折！"

### 第三板斧：A股映射（核心中的核心）

#### 产业链传导逻辑（必须准！）
```
[国际事件] → [技术/产品变化] → [中国产业链影响] → [A股具体标的]
```

**示例**：
- "马斯克说Optimus 2025年量产1000台" → 谐波减速器/丝杠需求放量 → 绿的谐波(688017)、双环传动(002472)
- "NVIDIA发布B200芯片" → 光模块速率升级 → 中际旭创(300308)、新易盛(300502)
- "DeepSeek API降价50%" → 推理算力需求激增 → 寒武纪(688256)、中科曙光(603019)

#### 映射类型
1. **直接供应链**：谁是苹果/特斯拉/英伟达的中国供应商？
2. **国产替代对标**：华为昇腾 vs NVIDIA、DeepSeek vs OpenAI
3. **产业链传导**：上游材料 → 中游制造 → 下游应用
4. **题材扩散**：龙头带动板块，板块带动指数
