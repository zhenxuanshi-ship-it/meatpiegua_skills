# Seedance 2.0 视频提示词完全手册：从新手到导演的实战指南

**写在开头**

第一次打开Seedance 2.0的时候，我盯着那个输入框发了十分钟呆。脑子里有一堆画面——雨夜 chase scene、仙侠对决、产品炫技展示——但真到打字的时候，却发现不知道怎么把它"翻译"成AI能听懂的话。

"一个女孩在海边散步"，生出来的东西确实美，但总觉得少了点什么。后来我才明白，Seedance 2.0不是那种"你说一句它就懂"的贴心助手，它更像是一个你雇来的专业摄影团队：导演、摄像、灯光、音效都配齐了，但你要是不把想要什么说清楚，他们就只能按"安全牌"来拍——好看，但平庸。

这篇手册是我用了两个月、生成了几百条视频后总结出来的。不是什么官方文档，就是实打实的经验——哪些写法能出好效果，哪些是浪费token的废话，怎么让AI真的拍出你想要的"电影感"。

## 一、先搞懂Seedance 2.0的脾气

### 它的强项是什么？

Seedance 2.0最厉害的不是"生成视频"，而是**多模态理解**。简单来说，它能同时处理：
- **画面**：你上传的图片、视频参考
- **动作**：复杂的运镜、角色动作
- **声音**：音效、配乐、甚至口型同步
- **时间**：15秒内的节奏把控、分镜切换

这意味着你写提示词的时候，也得从这几个维度去思考，而不是只描述"画面里有什么"。

### 它的理解优先级

根据我的测试，Seedance 2.0对提示词的理解大概是这个顺序：

1. **动作和运镜**（谁在动，镜头怎么动）
2. **场景和氛围**（在什么地方，什么感觉）
3. **参考素材**（@图片1、@视频1这些）
4. **声音设计**（音效、配乐）
5. **细节装饰**（光影、色调、风格词）

**关键原则**：把最重要的信息往前放，别在开头堆砌一堆形容词。

### 一个常见的误区

很多人以为写得越多越好，于是塞进去十几个元素："一个穿着红色风衣的女人，在雨夜的东京街头，手里拿着一把黑伞，身后有霓虹灯，远处有高铁驶过，地面有积水倒影，她停下脚步回头看，眼神忧郁......"

结果？AI可能只抓到了"女人"、"雨伞"、"霓虹灯"这三个词，其他的全丢了。

**我的经验**：一段提示词里，**核心要素不要超过5-6个**。想要复杂的画面？用分镜切开来写，别指望一句话讲完一部电影。

## 二、提示词的骨架：四段式写法

我总结了一个实用的结构，把提示词拆成四部分，按顺序写：

```
[技术规格] + [主体与动作] + [场景与运镜] + [声音与情绪]
```

### 第一部分：技术规格（一句话定调）

开头先把画面基调定下来，让AI知道你要什么风格：

```
2.35:1宽银幕，24fps，15秒，电影感纪实风格，高对比度冷暖对比
```

或者：

```
竖屏9:16，8秒，3D国漫CG风格，金红高饱和色调
```

**为什么要先写这个？** 因为这些参数会影响后面所有的画面决策。你要是先写内容再补风格，AI可能已经按"默认设置"生成了，改起来麻烦。

### 第二部分：主体与动作（必须具体）

别只说"一个人在走路"。要说：

```
一个穿着米色风衣的中年男人，双手插在口袋里，沿着石板路慢慢地走，偶尔抬头看看路灯，脚步在积水里踩出轻微的声响
```

**几个小技巧：**

**给人物贴标签**：年龄、穿着、神态、小动作。AI对"特征"很敏感。

❌ "一个女人"
✅ "三十岁左右的女人，披着凌乱的卷发，指甲涂着剥落的红色指甲油"

**动作要有节奏感**：用副词控制时间感知。

- "**慢慢地**抬起头" → 时间被拉长
- "**突然**转身" → 节奏变快
- "**犹豫了一下**，伸手去够" → 有停顿

**避免抽象动词**："感受"、"思考"、"回忆"这种词AI拍不出来。转化成外显动作：

❌ "他陷入了沉思"
✅ "他停下脚步，手指无意识地敲击着路灯杆，眼神放空"

### 第三部分：场景与运镜（导演视角）

场景不只是背景，它是情绪的放大器。同一个动作，放不同环境里，感觉完全不同：

| 环境 | 情绪暗示 |
|------|----------|
| 深夜的便利店里，荧光灯嗡嗡作响 | 孤独、都市感、失眠 |
| 阳光洒落的阁楼，灰尘在光柱里飘动 | 温暖、怀旧、时光静止 |
| 暴雨天的公交站台下，雨水顺着顶棚边缘流下 | 压抑、等待、被困住 |

**运镜怎么写？**

不要堆砌术语，按时间顺序描述镜头的运动：

```
开场是一个极远景，航拍视角俯瞰整个小镇，晨曦中的雾气笼罩着屋顶。然后镜头缓慢下摇，切换到跟随镜头，从背后拍摄主角沿着石板路行走。走到拐角处，镜头快速前推，切到特写——主角的手推开一扇木门。
```

看到没？**时间顺序 + 镜头运动 + 画面内容**，这样AI才能理解你的"分镜脚本"。

**常用运镜词汇（收藏备用）：**

| 类型 | 关键词 |
|------|--------|
| 景别 | 大远景、远景、全景、中景、近景、特写、大特写 |
| 运镜 | 推镜头、拉镜头、摇镜头、移镜头、跟拍、环绕拍摄、希区柯克变焦 |
| 角度 | 平视、俯拍、仰拍、鸟瞰视角、鱼眼镜头、第一人称视角 |
| 节奏 | 慢动作、快切、一镜到底、定格慢放、硬切 |

### 第四部分：声音与情绪（Seedance的杀手锏）

这是Seedance 2.0区别于其他工具的核心——它真的会给你配声音。

**声音设计包括：**
- **环境音**：雨声、风声、城市白噪音、咖啡馆人声
- **动作音效**：脚步声、关门声、纸张翻动、咖啡杯放下
- **配乐**：钢琴独奏、弦乐、电子音乐、爵士乐
- **人声**：对白、旁白、叹息、笑声

**写法示例：**

```
环境音是城市夜晚的白噪音，远处偶尔传来汽车驶过的闷响。当主角打开那封信时，背景音乐渐起，是大提琴的独奏，低沉而克制。读到动情处，音乐突然停止，只剩下纸张翻动的沙沙声和他压抑的呼吸。
```

**关键技巧**：声音要和画面同步。如果写"手指敲击键盘"，就要加上"键盘的敲击声清脆而有节奏"。如果只写画面不写声音，AI可能会给你配一段莫名其妙的人声吟唱。

## 三、十大能力的实战用法

Seedance 2.0有十大核心能力，每种对应不同的创作场景。下面是我在实战中总结的用法：

### 能力1：纯文本生成（最常用）

没有参考素材，纯靠文字描述。这是新手最常用的模式。

**提示词结构：**
```
(主体描述) + (动作序列) + (环境/光影) + (镜头语言) + (风格关键词)
```

**实战示例：**

```
2.35:1宽银幕，15秒，冷色调都市悬疑风格。

凌晨三点的地下停车场，一个穿着黑色连帽衫的女人快步走向她的车。镜头从侧面跟拍，她的脚步声在空旷的车库里回响。突然她停下脚步，回头张望——身后什么也没有。她松了口气，转身继续走。

镜头切换到她的背影全景，然后缓慢推近到肩部特写。她掏出钥匙，手指微微颤抖。钥匙插入锁孔的瞬间，停车场的灯突然全部熄灭。画面陷入黑暗，只有远处出口标志的绿光微弱地亮着。

环境音是脚步声、钥匙碰撞声、远处水管滴水的声音。灯灭的瞬间，音效戛然而止，只剩下她急促的呼吸声。
```

**效果：** 悬疑感拉满，画面节奏由慢到快再到突然的静默，情绪曲线完整。

---

### 能力2：@引用系统（一致性控制）

这是Seedance 2.0最实用的功能——上传参考图，让AI保持人物、产品、场景的一致性。

**官方命名规范：**
- 图片：`@图片1`、`@图片2`...`@图片9`
- 视频：`@视频1`、`@视频2`、`@视频3`
- 音频：`@音频1`、`@音频2`、`@音频3`

**写法格式：**
```
[角色]@图片N + [动作/剧情描述] + [场景]@图片N + [运镜/光影]
```

**实战示例——保持角色一致：**

```
15秒温情短片，电影感暖色调。

男人@图片1下班后疲惫地走在走廊，脚步变缓，最后停在家门口。脸部特写镜头，男人深呼吸，调整情绪，收起了负面情绪，变得轻松。然后特写翻找出钥匙，插入门锁。

门打开，室内光线温暖。他的小女儿和一只金毛犬欢快的跑过来迎接拥抱。室内布置温馨，墙上贴着儿童画。

全程自然的环境音：脚步声、钥匙声、门打开的吱呀声，然后是小女孩的笑声和狗叫声。背景音乐是淡淡的吉他曲，温暖治愈。
```

**关键技巧：**
- 必须在提示词里**明确说明每个素材的用途**：`@图片1的人物形象`、`参考@视频1的运镜效果`
- 如果不写清楚，AI可能会乱用或者忽略素材

---

### 能力3：运镜与动作精准复刻

上传一段参考视频，让AI学习里面的镜头语言、动作节奏。

**写法格式：**
```
参考@视频1的[运镜/动作/节奏] + [主体]@图片N + [场景描述]
```

**实战示例：**

```
参考@视频1的所有运镜效果和主角的面部表情节奏。

@图片1的男人站在@图片2的电梯中。电梯门缓缓关闭，他的表情从平静逐渐变成惊恐。参考@视频1的希区柯克变焦效果——背景压缩，人物脸部放大。然后几个环绕镜头展示电梯内视角。

电梯突然停住，灯光闪烁。门打开，跟随镜头男人走出电梯。电梯外场景参考@图片3的废弃医院走廊。男人环顾四周，呼吸急促。

环境音是电梯运行的机械声、灯光闪烁的电流声、门打开的金属摩擦声。当他走出电梯时，背景变成诡异的低频嗡鸣。
```

**效果：** AI会模仿参考视频的镜头运动方式，即使是复杂的多轴运动也能复刻。

---

### 能力4：首尾帧控制（变身/转场神器）

上传首帧图和尾帧图，AI会自动补全中间的过渡过程。这是做"变身"、"变装"、"场景转换"效果的最佳方式。

**写法格式：**
```
@图片1为首帧，@图片2为尾帧，[中间过渡描述]
```

**实战示例——古装变身：**

```
9秒，3D国漫CG风格，金红高饱和色调。

@图片1（现代女孩）为首帧，@图片2（古装女侠）为尾帧。

镜头环绕拍摄，女孩站在镜子前。一阵金色粒子从地面升起，围绕她旋转。她的现代服装逐渐消散，变成飘逸的红色古装。头发自动盘起，插上一支玉簪。

最后定格在尾帧画面：女侠手持长剑，目光坚毅，衣袍在能量波动中飘动。背景从现代房间变成古风宫殿。

音效：粒子升腾的电流声、衣服变换的沙沙声、最后一声清脆的剑鸣。
```

---

### 能力5：时间戳分镜法（15秒长视频的必杀技）

对于15秒的视频，最高级的写法是用时间戳精确控制每个镜头的内容。这是实际创作中最常用的技巧。

**写法格式：**
```
0-3秒：[画面描述 + 镜头语言]
4-8秒：[画面描述 + 镜头语言]
9-12秒：[画面描述 + 镜头语言]
13-15秒：[画面描述 + 镜头语言]
```

**实战示例——仙侠高燃战斗：**

```
15秒仙侠战斗镜头，金红暖色调，2.35:1宽银幕。

0-3秒：低角度特写，主角蓝袍衣摆被热浪吹得猎猎飘动。双手紧握雷纹巨剑，剑刃赤红电光持续爆闪。地面熔岩翻涌冒泡，远处魔兵嘶吼着冲锋逼近。主角低喝："今日，便以这柄剑，镇尔等邪祟！" 伴随剑鸣与熔岩咕嘟声。

4-8秒：环绕摇镜快切，主角旋身挥剑。剑刃撕裂空气迸射红色冲击波，前排魔兵被击飞碎裂成灰烬。伴随剑气破空声与魔兵惨嚎。

9-12秒：仰拍拉远定格慢放，主角跃起腾空。剑刃凝聚巨型雷光电弧劈向魔兵群，画面被金红色光芒充满。

13-15秒：缓推特写主角落地收剑的姿态，衣摆余波微动。冷声道："此界之门，不容踏越。" 音效收束为余音震颤与渐弱风声。
```

**为什么这样写效果好？**
- 时间戳让AI明确知道每个阶段该放什么内容
- 镜头语言（低角度、环绕摇镜、仰拍、缓推）精确控制画面感
- 台词和音效标注清楚，声画同步

### 能力6：视频延长（做长视频的核心技巧）

Seedance单次只能生成15秒，但可以用"视频延长"功能接龙。这是做30秒、60秒甚至更长视频的唯一方法。

**写法格式：**
```
将@视频1延长[X]s + [新增内容描述]
```

**关键原则：衔接点必须匹配**

上一段的结尾画面 = 下一段的开始画面

**实战示例——30秒咖啡广告（分两段）：**

**第1段（正常生成15秒）：**
```
15秒，竖屏9:16，电影感暖色调，咖啡广告风格。

0-5秒：特写，阳光透过百叶窗在木桌上形成条纹光影。一只咖啡杯放在桌上，热气缓缓升起。

6-10秒：镜头缓慢上摇，露出坐在桌前的年轻女人。她捧着杯子，闭眼闻了闻，露出满足的微笑。

11-15秒：她喝了一口，睁开眼睛看向镜头，眼神温柔。背景虚化，突出她的面部表情。

结尾定格画面：她手持咖啡杯，微笑着看向镜头。
```

**第2段（视频延长15秒）：**
```
将@视频1延长15秒。

0-5秒：接上段结尾画面，镜头缓慢拉远，展示整个咖啡馆环境。温暖的木质装修，窗外是城市街景。

6-10秒：切换到产品特写，咖啡杯上的logo清晰可见。咖啡表面有精美的拉花。

11-15秒：快速剪辑蒙太奇——咖啡豆倒入研磨机、热水冲泡、奶泡注入、最后呈现成品。配合轻快的爵士乐。

结尾画面：品牌slogan淡入："每一口，都是晨光。"
```

**衔接点检查：** 第1段结尾是"她手持咖啡杯微笑着"，第2段开头也是这个画面，完美衔接。

### 能力7：声音控制（口型同步、音色参考）

Seedance 2.0可以生成匹配的口型、参考音色、设计完整音效。

**写法格式：**
```
[画面描述] + 音色/旁白参考@视频1 + [台词内容用引号标注]
```

**实战示例——地产宣传片：**

```
15秒，2.35:1宽银幕，电影级写实风格地产纪录片。

航拍镜头俯瞰整个城市天际线，清晨的阳光洒在玻璃幕墙上。镜头缓缓下降，穿过云层，推进到一栋现代化写字楼。

镜头环绕建筑飞行，展示外立面设计。然后切到内部——大堂、电梯厅、办公室，每个空间都充满自然光。

旁白音色参考@音频1（沉稳男声）：
"在这里，每一步都是向上的力量。"

背景音乐：宏大的弦乐，逐渐加入钢琴点缀。环境音是城市的白噪音——远处车流、风声、偶尔的建筑回声。
```

### 能力8：一镜到底（长镜头连贯性）

生成不切换镜头的连续画面，从一个场景平滑过渡到另一个场景。

**写法格式：**
```
一镜到底 + @图片1@图片2@图片3... + [连续场景描述] + 全程不要切镜头
```

**实战示例——谍战追踪：**

```
15秒，谍战片风格，冷色调高对比。

@图片1为首帧画面。

一镜到底的跟拍镜头：穿着红风衣的女特工向前走。镜头正面跟拍，全景，不断有路人遮挡红衣女子。
```

## 四、不同场景的专用策略

### 场景1：电商/产品广告

**核心需求**：展示产品细节、质感、使用场景，配合品牌调性。

**提示词策略**：
- 多用特写和旋转展示
- 强调材质光影（金属反光、布料纹理、液体流动）
- 配合动感的背景音乐和卡点剪辑
- 可以加入3D分解、粒子特效等视觉效果

**实战示例——运动鞋广告：**

```
10秒，竖屏9:16，运动品牌广告风格，高能量电子配乐。

0-2秒：黑色背景，一道光柱打下，运动鞋从上方缓缓降落到画面中央，360度旋转展示。

3-5秒：快速剪辑——鞋底特写（展示缓震科技）、鞋面材质特写（透气网眼）、鞋带系统特写（快速系紧）。每个镜头配合重拍卡点。

6-8秒：场景切换，运动员穿着鞋子在城市街道上奔跑。镜头跟拍脚部，展示鞋子与地面的互动，水花四溅。

9-10秒：产品定格画面，品牌logo淡入，slogan："每一步，都是突破。"

音效：电子音乐重拍、跑步脚步声、城市环境音。禁止出现任何其他品牌logo或文字。
```

### 场景2：AI漫剧/仙侠

**核心需求**：视觉效果炸裂、动作流畅、特效华丽。

**提示词策略**：
- 指定3D国漫CG风格或中国风水墨风格
- 详细描述特效（法阵、能量波、粒子、剑气）
- 使用时间戳分镜法控制战斗节奏
- 台词要标注情绪和语气

**实战示例——修仙渡劫：**

```
15秒，3D国漫CG风格，金紫高饱和色调，史诗级特效。

0-4秒：仰拍，乌云密布的天空，紫色雷电在云层中翻滚。山峰之巅，白衣修士盘坐，周身环绕金色护体光芒。

5-8秒：第一道天雷劈下！修士双手结印，金光形成护盾。雷电与金光碰撞，能量波向四周扩散，树木被连根拔起。镜头环绕快切，展示360度战斗场景。

9-12秒：修士腾空而起，主动迎向雷云。他背后浮现巨大的法相虚影，手持仙剑，剑指苍穹。

13-15秒：万道雷霆凝聚成一条雷龙，咆哮着冲向他。修士眼中金光爆射，一剑斩出！画面被金紫色光芒充满，最后定格在他收剑入鞘的姿态，衣袍猎猎作响。

音效：雷鸣轰鸣、能量碰撞的爆炸声、仙剑出鞘的清鸣、修士的喝声："破！" 背景音乐：史诗级管弦乐，高潮处加入合唱。
```

### 场景3：短剧/对白场景

**核心需求**：情绪表达、对话自然、观众代入感强。

**提示词策略**：
- 画面和台词分开描述，台词标注角色和情绪
- 用正反打镜头切换对话双方
- 加入细微的肢体语言和表情变化
- 音效设计突出情绪转折

**实战示例——霸总追妻：**

```
15秒，横屏16:9，现代都市短剧风格，高对比度冷色调。

画面（0-5秒）：特写女主撕契约的动作，纸屑飘落。总裁单膝跪地伸手阻拦，眼神慌乱。女主侧身躲开，嘴角挂着冷漠笑意。

台词1（总裁，卑微慌乱）："苏晚！契约还没结束，你不能走！我给你钱，给你地位！"

画面（6-10秒）：女主抬脚避开他的手，将撕碎的契约纸扔在他脸上。镜头扫过周围宾客的窃窃私语和指指点点。

台词2（女主，冷漠反杀）："契约？顾总，当初是你说，我连给你提鞋都不配。现在求我？晚了！"

画面（11-15秒）：总裁僵在原地，脸上沾着纸屑，表情从震惊转为痛苦。女主转身昂首离开，红裙裙摆飘动，背影决绝。

音效：华丽又带张力的背景音，契约撕碎的清脆声响，宾客轻微的窃窃私语声。女主离开时，高跟鞋的声音格外清晰，每一步都是胜利的节奏。
```

### 场景4：科普/教学视频

**核心需求**：信息清晰、可视化强、专业但不枯燥。

**提示词策略**：
- 使用4K医学CGI或科学可视化风格
- 半透明、剖面、放大等展示手法
- 配合教育性旁白，语速适中
- 关键信息用动画标注或高亮

**实战示例——人体免疫系统：**

```
15秒，4K医学CGI风格，科学教育纪录片质感。

0-4秒：半透明人体模型，镜头推进到胸腔。血液流动效果，红细胞在血管中穿行。

5-8秒：特写一个病毒颗粒侵入细胞的过程。然后白细胞从四面八方赶来，包围并吞噬病毒。动画标注："T细胞识别抗原"。

9-12秒：展示抗体与病毒结合的过程，病毒被标记并被消灭。镜头拉远，展示整个免疫系统的协同工作。

13-15秒：回到完整人体，免疫系统发光高亮。旁白："你的免疫系统，24小时守护健康。"

旁白音色：温和专业的女声，语速适中。背景音乐：轻快的电子音乐，不抢旁白风头。音效：血液流动的细微声响、细胞活动的低频音效。
```

### 场景5：MV/音乐卡点视频

**核心需求**：画面与音乐完美同步、节奏感强、视觉冲击力。

**提示词策略**：
- 指定画幅比和帧率（通常是2.35:1，24fps）
- 分镜头描述每个镜头的场景、动作、时长
- 强调声音设计与节拍同步
- 多图卡点参考视频节奏

**实战示例——街舞MV片段：**

```
8秒，2.35:1，24fps，街舞MV风格，霓虹高饱和冷暖对比。

@图片1@图片2@图片3@图片4@图片5，参考@视频1的画面节奏和卡点。

每0.8秒切换一个画面，配合鼓点重拍。画面中舞者的动作要有爆发力——定格在踢腿、旋转、freeze的瞬间。

镜头语言：快速剪辑、鱼眼镜头变形、环绕拍摄、低角度仰拍。浅景深突出舞者动作，背景虚化形成光斑。

声音设计优先：舞步声、鞋底摩擦地板的吱吱声、呼吸声、衣料摩擦声必须清晰并与节拍贴合。

禁止任何文字、logo、水印。
```

## 五、避坑指南：这些写法会让你的视频翻车

### 坑1：矛盾的描述

❌ "画面要明亮但又要昏暗的氛围"
❌ "人物既开心又难过"

AI会懵的。情绪可以是复杂的，但不能是矛盾的。要说"强颜欢笑"、"笑着笑着突然沉默"——这是情绪的转折，不是矛盾。

### 坑2：过于抽象的概念

❌ "表现出存在主义的焦虑"
❌ "呈现时间的流逝感"

把这些翻译成具体的画面元素：
- "存在主义焦虑" → "凌晨三点的便利店里，一个穿着西装的男人对着关东煮发呆，热气模糊了他的眼镜"
- "时间的流逝" → "同一扇窗户，从春天开花到冬天落雪的快速剪辑，窗台上日历翻页"

### 坑3：忽视@引用的清晰度

上传了素材但不在提示词里明确说明用途，AI可能会忽略或者乱用。

❌ "用这张图片生成视频"
✅ "@图片1的人物形象作为主角，@图片2的场景作为背景"

### 坑4：超出平台限制

- **不要上传写实真人脸部素材**（会被平台拦截）
- 单次生成不超过15秒（需要长视频用延长功能分段）
- 混合素材最多12个文件（图片+视频+音频合计）

### 坑5：动作过于复杂

Seedance 2.0在复杂连续动作上还有局限。不要写：

❌ "她优雅地转身，同时抛出披风，披风在空中划出一道弧线，然后稳稳地落在椅背上，她顺势坐下"

分解成简单的动作，用分镜切换：

✅ "她转过身，披风随风扬起（画面1）。镜头切到下一个画面，披风已经搭在椅背上，她正在坐下（画面2）"

## 六、私房技巧（进阶玩家专用）

### 技巧1：用"电影感"兜底

当你不知道怎么形容光影时，直接说"电影感的光影"、"电影感的色调"，Seedance 2.0一般都能给你一个不错的结果。这算是它的"隐藏指令"。

### 技巧2：指定导演风格

想要某种特定的视觉风格？直接提导演名字：

- "类似**王家卫**的风格" → 迷离的光影、抽帧、霓虹灯、慢快门
- "类似**诺兰**的风格" → IMAX画幅、实景拍摄感、宏大叙事、低音轰鸣
- "类似**是枝裕和**的风格" → 日常生活感、柔和自然光、家庭场景、安静
- "类似**扎克·施奈德**的风格" → 高对比度、慢动作、史诗感

### 技巧3：用括号强调重点

当某个元素特别重要时，用括号强调：

```
主角是一个（穿着红色大衣在灰蒙蒙的人群中特别显眼）的女人
```

### 技巧4：控制"时间感知"

Seedance 2.0可以通过提示词来控制"感觉上"的时长：

- "**漫长的**等待" → AI会放慢节奏，增加停顿
- "**一闪而过的**记忆" → AI会加快节奏，快速剪辑
- "**时间仿佛静止了**" → AI会用慢动作或定格

### 技巧5：重复关键词强化

想让某个元素特别突出？在不同部分重复提到：

```
雨一直在下。雨声是这个场景的主角。镜头上有雨水滑落。地面上的积水倒映着霓虹。一切都湿漉漉的。
```

## 七、完整案例：从想法到成片

### 案例：我想拍一个"深夜便利店的邂逅"

**第一步：确定情绪基调**
不是那种浪漫的偶像剧，而是都市人的孤独与偶然的连接——客气、疏离、但有一丝温暖。

**第二步：设计场景和角色**
- 场景：24小时便利店，凌晨2点，荧光灯嗡嗡作响
- 角色A：加班到深夜的上班族，疲惫，穿着皱巴巴的衬衫
- 角色B：便利店店员，无聊地刷手机，偶尔抬头看监控

**第三步：构建镜头（使用时间戳）**

**第四步：加入声音设计**
- 环境音：冰箱嗡嗡声、空调运转声、远处街道的车流
- 动作音效：门打开的叮咚声、微波炉的叮声、撕包装袋的声音
- 背景音乐：若有若无的钢琴，孤独但温暖

**完整提示词：**

```
15秒，2.35:1宽银幕，电影感纪实风格，冷色调中带一丝暖光。

0-4秒： establishing shot，便利店全景，透过玻璃门拍摄。店内空无一人，荧光灯发出嗡嗡的声响。一个男人推门进来，门铃发出清脆的"叮咚"声。

5-8秒：跟随镜头，男人走到货架前拿了一个饭团和罐装咖啡。镜头切换到收银台视角，店员抬头看了他一眼，又低头继续刷手机。

9-12秒：男人站在微波炉前加热饭团，盯着旋转的盘子发呆。微波炉"叮"的一声，他愣了一下才反应过来。这个细节特写。

13-15秒：他坐在窗边的位子上，撕开饭团的包装。镜头从窗外拍摄，他的侧脸被店内的灯光照亮，窗外的街道一片漆黑。他咬了一口饭团，嘴角微微上扬——这是今天第一个真心的表情。

环境音：冰箱的嗡嗡声、空调运转声、远处街道偶尔驶过的汽车。微波炉的叮声格外清脆。背景音乐是淡淡的钢琴曲，像是一个人深夜的独白。最后画面定格，钢琴声慢慢淡出，只剩下便利店的白噪音。
```

## 八、结语

写Seedance 2.0的提示词，本质上是在学习**用视觉和声音讲故事**。这不是什么高深的技术，就是导演的基本功——知道什么时候该特写，什么时候该全景；知道怎么用光影说台词没说的话；知道音乐该什么时候进来，什么时候停掉。

最好的学习方式就是多看多试。看那些让你印象深刻的电影片段，试着用文字把它们描述出来，然后看看Seedance 2.0能给你什么。慢慢地，你就会找到自己的"语言"——那种你一写，AI就懂的默契。

记住，AI是工具，你是导演。它负责执行，你负责想象。

现在，去拍点什么吧。
